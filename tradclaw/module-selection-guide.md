# Module Selection Guide

Use the interview to recommend only the modules that fit.

**Single source for “good defaults”:** the *Good defaults* section below is canonical. The repository [`README.md`](../README.md) points here so starter recommendations do not drift.

## Core modules

### 1. Calendar briefs
Enable if the household has:
- multiple calendars
- pickup complexity
- recurring activities
- frequent schedule confusion

Includes:
- morning brief
- afternoon pickup check
- weekend preview

### 2. School triage
Enable if there are:
- school emails or apps to watch
- forms, deadlines, theme days, field trips
- frequent missed notices

### 3. Meal planning and shopping
Enable if the household wants:
- dinner planning help
- one place for grocery and household items
- pantry-gap prompts

### 4. Homework and learning log
Enable if the household wants:
- homework photos tracked
- schoolwork summarized
- recurring struggles or strengths remembered

### 5. Book inventory
Enable if the household has:
- lots of children’s books
- a desire for easy recommendations from owned books
- a habit of buying duplicates by accident

### 6. Home maintenance
Enable if the household wants:
- recurring reminders for filters, resets, seasonal cleanup, donation runs, supply checks

### 7. Helper payments
Enable if the household has:
- recurring paid helpers
- irregular or easy-to-forget payment cadence

### 8. Custom stories
Enable if the household wants:
- bedtime stories
- personalized stories for kids
- educational or themed stories
- optional voice/TTS story mode

## Good defaults

For most households, start with:
- calendar briefs
- school triage
- meal planning and shopping
- home maintenance

Then add the rest if clearly wanted.
