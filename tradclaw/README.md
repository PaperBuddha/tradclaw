# Tradclaw — opinionated setup layer

This folder is the **scaffold / runbook** for turning the Tradclaw example into *your* household workspace. It mirrors the idea of a separate “policy + setup” layer (similar in spirit to [clawchief](https://github.com/snarktank/clawchief): source-of-truth guidance next to a copyable workspace).

## Start here

1. **[`BOOTSTRAP.md`](BOOTSTRAP.md)** — first-run instructions for the assistant (read order + post-interview actions).  
2. **[`onboarding-interview.md`](onboarding-interview.md)** — interview question bank.  
3. **[`module-selection-guide.md`](module-selection-guide.md)** — when to enable each capability.  
4. **[`apply-interview-results.md`](apply-interview-results.md)** — what to produce after the interview and which files to tailor.  
5. **[`openclaw-primitives.md`](openclaw-primitives.md)** — how Tradclaw maps to real OpenClaw workspace files (links to [docs.openclaw.ai](https://docs.openclaw.ai/concepts/agent-workspace)).  
6. **[`example-first-run.md`](example-first-run.md)** — optional short tone sample for a first conversation.

The **household files to copy and edit** live one level up in [`workspace/`](../workspace/).
