# Skills (optional modules)

Each folder is an **optional** capability. None are required for a minimal household setup.

After [`tradclaw/BOOTSTRAP.md`](../tradclaw/BOOTSTRAP.md) and [`tradclaw/module-selection-guide.md`](../tradclaw/module-selection-guide.md), copy or register only the skills the household wants.

| Skill | Typical use |
|-------|----------------|
| `book-inventory` | Track owned books and wishlists |
| `custom-stories` | Bedtime / personalized stories |
| `helper-payments` | Recurring cleaner, sitter, etc. |
| `home-maintenance` | Filters, seasonal resets, schedules |
| `homework-log` | Worksheet photos → concise log |
| `meal-planner` | Weekly plan + shopping alignment |
| `school-triage` | Forms, theme days, notices |

Paths in skill files refer to **`workspace/resources/...`** while this repo is intact, or **`resources/...`** after the user copies `workspace/` to their OpenClaw household root.
