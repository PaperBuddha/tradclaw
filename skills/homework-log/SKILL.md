# Homework Log Skill

Use this skill when the user wants to:
- log homework from photos
- track what a child is learning
- remember topics that need more practice
- build a light educational record over time

## Goal

Create a useful running log of schoolwork without turning the child into a data project.

## Inputs

Usually one or more photos of worksheets, writing, math pages, or project instructions.

## Workflow

1. Identify:
   - child if known
   - date
   - subject area
   - assignment or worksheet theme
   - skills being practiced
   - visible struggles or wins
2. Add a concise entry to the relevant file in `workspace/resources/homework/` (or `resources/homework/` under a deployed `workspace/` copy).
3. If there is a recurring weak spot, add it to `workspace/resources/homework/concepts-to-reinforce.md`.
4. If asked, summarize progress over time.

## Entry format

- 2026-04-13 — Math — double digit addition with regrouping
  - practiced: carrying/regrouping, equation layout
  - seemed hard: alignment across columns
  - note: needed reminders to slow down

## Rules

- Be descriptive, not judgmental.
- Never overstate difficulty from a single worksheet.
- Focus on patterns over isolated mistakes.
- Protect privacy if sharing outside the family context.

## Useful follow-ups

- suggest 2 or 3 reinforcement ideas
- note likely vocabulary or lesson themes
- remind the household what to review before a test or quiz
