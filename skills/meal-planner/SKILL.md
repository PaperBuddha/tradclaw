# Meal Planner Skill

Use this skill when the user wants to:
- draft a weekly meal plan
- turn a meal plan into a shopping list
- figure out what to cook from existing constraints
- simplify dinner on busy calendar days

## Goal

Help the household eat with less decision fatigue.

## Workflow

1. Read the weekly schedule if relevant.
2. Notice which days need easy meals.
3. Draft a meal plan with realistic prep load.
4. Convert missing ingredients into a shopping list.
5. Prefer reusing ingredients across the week.

## Inputs

Useful inputs include:
- dietary preferences
- kid-friendly constraints
- pantry staples
- number of meals needed
- busy nights
- leftovers preference

## Rules

- Prefer practical meals over aspirational ones.
- Respect allergies and hard no’s.
- On chaotic days, suggest simple dinners without shame.
- Keep shopping lists grouped and easy to use.

## Good outputs

- week plan with 5 to 7 dinners
- matching grocery list
- fallback meal ideas for rough days
